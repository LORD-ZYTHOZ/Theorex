  <behavior>
    extractConcepts cases:
    - input: "TypeScript is a superset of JavaScript" → output includes RawConcept with surfaceForm containing "TypeScript" and "JavaScript" (proper nouns)
    - input: "Barack Obama visited New York" → output includes entries for "Barack Obama" (Person, isMultiWord=true) and "New York" (Place, isMultiWord=true)
    - input: "the the the" → output is empty [] (no noun phrases, stop words only)
    - input: same text twice → output is structurally identical both calls (pure function)
    - input: text with "dogs are animals" → deduplication: only one entry per unique lowercased surface form
    - compromise .json() returns any[] → validated before use (missing text/tags fields produce empty result, not crash)

    normalizeConcepts cases:
    - input: RawConcept with surfaceForm "developers" → canonicalForm "developer" (plural → singular)
    - input: RawConcept with surfaceForm "running" → canonicalForm "run" (gerund → base)
    - input: RawConcept with surfaceForm "TypeScript" → canonicalForm "typescript" (lowercased)
    - input: RawConcept with empty surfaceForm → canonicalForm falls back to lowercased surfaceForm (no crash)
    - input: same concepts twice → output canonical forms are identical (pure)
  </behavior>
  <implementation>
    src/extract.ts:
    - Import nlp from "compromise" and RawConcept from "../src/types.ts"
    - Write a parseCompromiseJson() internal validator that takes unknown[] and returns Array<{text: string; tags: string[]}> — use `unknown` cast, check for `text: string` and `tags: string[]` fields, skip malformed entries silently (log nothing)
    - extractConcepts() calls nlp(text).nouns().json() and nlp(text).topics().json() — both go through parseCompromiseJson()
    - Deduplication: use a Set<string> keyed on surfaceForm.trim().toLowerCase()
    - topics() covers .people(), .places(), .organizations() — overlap with nouns() is handled by deduplication
    - Map to RawConcept: { surfaceForm: c.text.trim(), tags: c.tags ?? [], isMultiWord: c.text.trim().includes(" ") }
    - Return as readonly array (Object.freeze not required — TypeScript readonly is sufficient for Phase 0)
    - Filter out entries where surfaceForm is empty after trim

    src/normalize.ts:
    - Import nlp from "compromise" and { RawConcept, NormalizedConcept } from "../src/types.ts"
    - normalizeConcepts() maps each RawConcept to NormalizedConcept
    - For each concept: call nlp(c.surfaceForm).normalize({ plurals: true, verbs: true, case: true, acronyms: true }).text().toLowerCase().trim()
    - If result is empty string: fall back to c.surfaceForm.toLowerCase().trim()
    - Spread c into new object, add canonicalForm — never mutate input (immutability rule)
    - File must stay under 50 lines
