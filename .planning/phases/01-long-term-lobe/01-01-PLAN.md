<feature>
  <name>AxonStore — Graphology-backed concept web store</name>
  <files>src/axon/store.ts, tests/axon/store.test.ts</files>
  <behavior>
    AxonStore wraps UndirectedGraph with typed node/edge attributes.

    Expected behaviors (each becomes a test):
    - mergeNode(event) where event is a fresh ConceptEvent → node created with correct attrs including sentiment_tier: "NEUTRAL", relevance_tier: "ACTIVE", string key = String(concept_id)
    - mergeNode(event) where node already exists → frequency_count incremented, last_seen updated, other attrs unchanged
    - mergeEdge(idA, idB) first call → edge created with strength: 0.1, co_occurrence_count: 1
    - mergeEdge(idA, idB) second call → strength increases by 0.05, co_occurrence_count = 2, last_co_occurrence updated
    - mergeEdge strength clamps at 1.0 — never exceeds maximum
    - save(path) → writes to path+".tmp" then renames atomically to path
    - load(path) → returns graph with typed attributes (UndirectedGraph.from<AxonNodeAttrs, AxonEdgeAttrs>)
    - save then load → node count and attributes preserved byte-for-byte through JSON serialization
    - graph.hasNode(String(numericId)) → true after mergeNode with numeric concept_id
    - new AxonStore() starts with empty graph (0 nodes, 0 edges)

    Key invariants:
    - NEVER call graph.addNode() — always graph.mergeNode()
    - NEVER call graph.addEdge() — always graph.mergeEdge()
    - Node keys are always String(concept_id) — never the raw number
    - Collect neighbors before mutating (forEachNeighbor mutation is unsafe — use graph.neighbors() then iterate)
  </behavior>
  <implementation>
    AxonNodeAttrs and AxonEdgeAttrs interfaces defined in store.ts (these are Phase 1 types, not in src/types.ts).

    AxonNodeAttrs:
    ```typescript
    interface AxonNodeAttrs {
      concept_id: number;
      surface_form: string;
      importance_weight: number;
      relevance_tier: "ACTIVE" | "MILD" | "LESS";
      sentiment_tier: "PREFERRED" | "NEUTRAL" | "DISPREFERRED";
      last_seen: string;       // ISO 8601
      frequency_count: number;
      source_weight: number;
    }
    ```

    AxonEdgeAttrs:
    ```typescript
    interface AxonEdgeAttrs {
      strength: number;           // 0.0–1.0
      co_occurrence_count: number;
      last_co_occurrence: string; // ISO 8601
    }
    ```

    AxonStore class methods:
    - `mergeNode(event: ConceptEvent): string` — upserts node from ConceptEvent; returns node key
    - `mergeEdge(idA: number, idB: number, timestamp: string): void` — upserts/strengthens edge
    - `async save(path: string): Promise<void>` — atomic write via Bun.write + node:fs/promises rename; path+".tmp" always in same directory
    - `static async load(path: string): Promise<AxonStore>` — loads from axon.json if exists; returns empty store if not found (ENOENT)
    - `get graph(): UndirectedGraph<AxonNodeAttrs, AxonEdgeAttrs>` — read-only access to underlying graph for iteration

    File does NOT export: scoring logic, propagation, pruning — those live in separate modules (Plans 03-04).

    Atomic write: `import { rename } from "node:fs/promises"`. Tmp path = `path + ".tmp"`. ALWAYS same-directory to ensure same filesystem (avoid EXDEV).

    Use `UndirectedGraph.from<AxonNodeAttrs, AxonEdgeAttrs>(raw)` not `Graph.from()` — typed constructor required.
  </implementation>
</feature>
