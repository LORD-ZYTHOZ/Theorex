  <behavior>
    parseMemory() behavior:
    - Splits raw string by "\n", identifies lines starting with "## "
    - Everything before first "## " line goes to preamble
    - Each "## " line starts a new section; its rawBody is all lines until next "## " joined by "\n"
    - Last section's rawBody ends with "\n" (the trailing newline is part of the body, not added separately)

    serializeMemory() behavior:
    - Joins preamble + sections each as `heading + "\n" + rawBody`
    - Sections joined with "\n" between them
    - Result must equal original raw string exactly (byte-identical)

    CRITICAL PITFALL (from RESEARCH.md):
    - The final section's rawBody already ends with "\n". When reconstructing, join as:
      `parts.push(section.heading + "\n" + section.rawBody)`
    - Then `parts.join("\n")` adds a "\n" between sections, so between section N and N+1 there's one "\n" (from join)
    - Wait — this would double the newline. The correct approach:
      The preamble ends with "\n\n". Each section is `heading + "\n" + rawBody` where rawBody ends with "\n\n" (blank line before next heading). The join separator must be "" (empty), not "\n".
    - Implementation strategy: `return [preamble, ...sections.map(s => s.heading + "\n" + s.rawBody)].join("")`
    - Verify this produces byte-identical output by running the actual MEMORY.md file through parser in tests.

    writeMemoryAtomic() behavior:
    - Accepts targetPath and content string
    - Writes to targetPath + ".tmp" via Bun.write
    - Renames tmp to target via node:fs/promises rename (atomic on same filesystem)
    - Never writes directly to targetPath

    readMemory() behavior:
    - Reads file at path and returns raw string
    - Returns empty string if file does not exist (ENOENT — cold start)

        last_classified: string;  // ISO 8601
      }>;
    }
    ```
    readMeta() returns default TheorexMeta if file absent. writeMeta() uses same atomic write pattern.
  </behavior>

    export interface MemorySection {
      readonly heading: string;
      readonly rawBody: string;
    }
    export interface ParsedMemory {
      readonly preamble: string;
      readonly sections: readonly MemorySection[];
    }

    export function parseMemory(raw: string): ParsedMemory {
      const lines = raw.split("\n");
      const sections: MemorySection[] = [];
      let preambleLines: string[] = [];
      let currentHeading: string | null = null;
      let currentBodyLines: string[] = [];

      for (const line of lines) {
        if (line.startsWith("## ")) {
          if (currentHeading !== null) {
            sections.push({ heading: currentHeading, rawBody: currentBodyLines.join("\n") });
          } else {
            preambleLines = [...preambleLines, ...currentBodyLines];
          }
          currentHeading = line;
          currentBodyLines = [];
        } else {
          currentBodyLines.push(line);
        }
      }
      if (currentHeading !== null) {
        sections.push({ heading: currentHeading, rawBody: currentBodyLines.join("\n") });
      } else {
        preambleLines = [...preambleLines, ...currentBodyLines];
      }

      return {
        preamble: preambleLines.join("\n"),
        sections,
      };
    }

    export function serializeMemory(parsed: ParsedMemory): string {
      return [
        parsed.preamble,
        ...parsed.sections.map((s) => s.heading + "\n" + s.rawBody),
      ].join("");
    }
    ```
