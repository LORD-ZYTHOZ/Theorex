<feature>
  <name>Scan (re-score) and prune (archive + delete) operations</name>
  <files>src/axon/scan.ts, src/axon/prune.ts, tests/axon/scan.test.ts, tests/axon/prune.test.ts</files>
  <behavior>
    scanAxon(axonPath: string, config: Config, nowMs?: number): Promise<void>
    - Load AxonStore from axonPath (or return early if file doesn't exist — cold start)
    - For every node: compute neighborStrengths (collect edge strengths for all neighbors), compute compositeScore, update importance_weight and relevance_tier on node
    - Collect nodes before iterating (avoid mutation during forEachNode)
    - For every edge: apply exponential decay to strength using same formula as recency decay with halfLifeDays. Drop edges where strength < config.edgePruneThreshold (0.01) via graph.dropEdge()
    - Write updated graph atomically via store.save(axonPath)
    - Does NOT touch MEMORY.md

    pruneAxon(axonPath: string, archiveDir: string, config: Config, nowMs?: number): Promise<void>
    - Load AxonStore from axonPath
    - Identify LESS nodes where last_seen is more than config.pruneThresholdDays ago (30 days default)
    - For each pruned node: serialize node attributes to a JSONL record
    - Ensure archiveDir exists (mkdir -p via node:fs/promises mkdir with { recursive: true })
    - Write all pruned records to archiveDir/pruned-{timestamp}.jsonl as a JSONL file (one JSON object per line)
    - Drop each pruned node from graph via graph.dropNode() — ONLY after JSONL write succeeds
    - Prune edges whose both endpoints are removed automatically by dropNode (Graphology behavior)
    - Write updated graph atomically via store.save(axonPath)
    - Log pruned count to console

    Edge decay in scanAxon:
    - Use same exponential decay formula as recency: newStrength = strength × Math.exp(-Math.LN2 / halfLifeDays × daysSinceCoOccurrence)
    - daysSinceCoOccurrence = (nowMs - new Date(edge.last_co_occurrence).getTime()) / 86_400_000
    - If newStrength < config.edgePruneThreshold → drop edge entirely

    JSONL archive record format:
    ```typescript
    interface ArchiveRecord {
      archived_at: string;     // ISO 8601 of when prune ran
      concept_id: number;
      surface_form: string;
      importance_weight: number;
      last_seen: string;
      frequency_count: number;
      source_weight: number;
      relevance_tier: "LESS";
      sentiment_tier: "PREFERRED" | "NEUTRAL" | "DISPREFERRED";
    }
    ```
    Written as: `record1JSON\nrecord2JSON\n` (newline-delimited, trailing newline)
  </behavior>
  <implementation>
    scan.ts (~90 lines):
    - Import: AxonStore, compositeScore, classifyTier, Config, recencyScore
    - Collect all node keys first: `const nodeKeys = store.graph.nodes()`
    - Collect all edge keys first: `const edgeKeys = store.graph.edges()`
    - Mutate after collection — no mutation during iteration
    - Edge decay uses same Math.exp formula as node recency

    prune.ts (~80 lines):
    - Import: AxonStore, Config from "../config", mkdir from "node:fs/promises"
    - Archive before drop is the invariant: if JSONL write throws, graph is not modified
    - Use try/finally pattern: write archive → if succeeds → drop nodes
    - JSONL: `lines.join("\n") + "\n"` — trailing newline on last record

    PM2 ecosystem config (Claude's discretion — create ecosystem.config.cjs):

    - File: `ecosystem.config.cjs` at project root
    - Content: minimal PM2 cron config that runs `bun run src/cli/index.ts scan` on cron `0 */6 * * *`
  </implementation>
