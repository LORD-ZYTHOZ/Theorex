  <name>Task 1: Implement CLI entry point (all four commands)</name>
  <files>src/cli/index.ts</files>
  <action>
    Create `src/cli/index.ts` — the CLI entry point using Bun.argv and util.parseArgs.

    Structure:
    ```typescript
    import { parseArgs } from "util";
    import { scanAxon } from "../axon/scan";
    import { pruneAxon } from "../axon/prune";
    import { propagateActivation } from "../axon/propagate";
    import { AxonStore } from "../axon/store";
    import { loadConfig } from "../config";

    const AXON_PATH = "data/axon.json";
    const ARCHIVE_DIR = "data/archive";

    const { positionals } = parseArgs({
      args: Bun.argv.slice(2),
      allowPositionals: true,
      strict: false,
    });

    const [subcommand, ...rest] = positionals;
    const config = await loadConfig();

    switch (subcommand) {
      case "scan":
        await scanAxon(AXON_PATH, config);
        console.log("Scan complete.");
        break;

      case "status":
        // Load store, iterate nodes, print table
        break;

      case "ref": {
        const keyword = rest[0];
        if (!keyword) {
          console.error("Usage: theorex ref <keyword>");
          process.exit(1);
        }
        // Load store, find or create node by keyword (fuzzy match surface_form), propagate, save
        break;
      }

      case "prune":
        await pruneAxon(AXON_PATH, ARCHIVE_DIR, config);
        console.log("Prune complete.");
        break;

      default:
        console.error(`Unknown command: ${subcommand ?? "(none)"}`);
        console.error("Usage: theorex <scan|status|ref <keyword>|prune>");
        process.exit(1);
    }
    ```

    `status` command implementation:
    - Load AxonStore from AXON_PATH
    - Collect all nodes as array: `{ key, attrs }` objects
    - Sort by importance_weight descending
    - Print header: `"Concept Web — ${nodeCount} nodes"`
    - Print table rows with columns: concept_id, surface_form, relevance_tier, sentiment_tier, importance_weight (2 decimal places), frequency_count, last_seen (date only)
    - Use console.table or manual column alignment with padEnd (Claude's discretion — pick whichever produces clean output)
    - Empty graph → print "No concepts in axon. Run: theorex scan"

    `ref` command implementation:
    - Load AxonStore
    - Search for node where surface_form.toLowerCase() === keyword.toLowerCase() OR String(concept_id) === keyword
    - If not found: print error "Concept not found: <keyword>. Use theorex scan to ingest concepts first." and exit 1
    - If found: call propagateActivation(store, nodeKey, 1.0, Date.now())
    - Save store atomically to AXON_PATH
    - Print: "Referenced: <surface_form> (tier: <relevance_tier>)"

    All paths are relative to process.cwd() — no hardcoded absolute paths. CLI is meant to be run from the project root.

    Keep file under 120 lines. No business logic here — just dispatch and I/O.
