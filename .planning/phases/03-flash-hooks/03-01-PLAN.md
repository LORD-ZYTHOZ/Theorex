**Constants:**
```typescript
const FLASH_DIR = "data/flash";
const MAX_EVENTS = 50;    // FLH-01
const TOKEN_CEILING = 4000; // FLH-03
```

**estimateTokens(events: readonly FlashEvent[]): number**
- Returns Math.ceil(JSON.stringify(events).length / 4)
- Pure function, no I/O

**enforceRingBuffer(events: readonly FlashEvent[], incoming: FlashEvent): readonly FlashEvent[]**
- Appends incoming to events
- Slices to last MAX_EVENTS (ring eviction)
- Then trims oldest until estimateTokens < TOKEN_CEILING OR only 1 event remains (never evict the last event — keeps minimum of 1)
- Returns new array (never mutates input)

**readFlash(sessionId: string, dir?: string): Promise<FlashBuffer>**
- Reads data/flash/{sessionId}.json
- If file not found (ENOENT), returns { session_id: sessionId, events: [] }
- Uses node:fs/promises readFile, parses JSON
- Never throws on missing file

**writeFlash(buffer: FlashBuffer, dir?: string): Promise<void>**
- Writes to data/flash/{buffer.session_id}.json
- ATOMIC: write to tmpdir() temp file, then rename() to final path (same pattern as Phase 1 LTM-04)
- Uses node:fs/promises: mkdir (recursive), writeFile, rename
- Uses import { tmpdir } from "node:os"
- Uses node:fs/promises, NOT Bun.write (to stay consistent with atomic rename pattern)
- Never mutates the buffer
