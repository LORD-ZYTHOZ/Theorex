Create src/moments/search.ts:

```typescript
// src/moments/search.ts — BM25 search over moment stories.
// Uses same wink-bm25-text-search CJS interop pattern as src/short-term/bm25.ts.
// Minimum 3-doc sentinel padding applied when moments.length < 3.

import { createRequire } from "node:module";
const require = createRequire(import.meta.url);
const winkBm25 = require("wink-bm25-text-search");

export interface MomentSearchResult {
  readonly id: string;
  readonly story: string;
  readonly score: number;
  readonly timestamp: string;
}
```

buildMomentBm25Index(moments):
- Pad moments array with sentinel objects (id: "", story: "", etc.) when moments.length < 3
- Create engine via winkBm25()
- defineConfig: { fldWeights: { story: 3 }, bm25Params: { k1: 1.2, b: 0.75 } }
- definePrepTasks: ["tokenize", "stem"] on "story" field
- For each moment (including sentinels): engine.addDoc({ story: moment.story }, index)
- engine.consolidate()
- Return engine

searchMoments(moments, query, limit):
- If moments empty: return []
- Build index, run engine.search(query, limit * 2)
- Map results back to MomentSearchResult using the index mapping
- Filter out sentinel results (id === "")
- Sort descending by score, return first limit results
