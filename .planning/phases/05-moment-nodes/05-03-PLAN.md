**Part B: src/moments/capture.ts** — implement three exported functions:

captureCodeRefs(): Promise<CodeRef[]>
- Run `Bun.$\`git diff --name-only HEAD\`` (Bun shell API)
- Parse stdout: split by newline, filter empty, map to { file: line.trim(), line: 1 }
- Return [] on ANY failure (try/catch wraps everything — non-git dirs, empty diff)

extractConceptIds(story: string, knownIds: Set<number>): number[]
- Call processText({ text: story, sourceWeight: 1.0, nodeType: "moment", timestamp: new Date().toISOString() })
- Map events to concept_ids, filter by knownIds.has(id)
- Return unique IDs (dedup via Set)
- Return [] if story is empty or processText returns nothing

runMoment(story, axonPath, config, explicitRefs?, momentsDir?):
- Validate story not empty (throw if empty — CLI already guards this)
- Load AxonStore from axonPath
- Build knownIds: new Set(axon.graph.nodes().map(k => axon.graph.getNodeAttributes(k).concept_id as number))
  NOTE: Graphology stores concept_id as a node attribute, not the key. Use getNodeAttributes(key).concept_id
- Call extractConceptIds(story, knownIds)
- Call captureCodeRefs() — merge with explicitRefs (explicit takes priority; concat both, dedup by file:line)
- Build MomentNode: { id: crypto.randomUUID(), timestamp: new Date().toISOString(), story, code_refs: mergedRefs, concept_ids }
- Call createMoment(moment, momentsDir ?? config.momentsDir)
- console.log(`Moment saved: ${moment.id}`)
